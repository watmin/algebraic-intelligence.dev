## Read the series

Two tracks. Read in either order — the primers are reference material; the story is chronological.

**Primers** — the technical foundation, bottom-up:

- [VSA/HDC: A Working Introduction](/blog/primers/series-001-000-vsa-primer/) — what a hypervector is, the three operations, why it works for structured data
- [Atoms, Vectors, and the Encoding Stack](/blog/primers/series-001-001-atoms-and-vectors/) — how structured data becomes geometry; the hash-function codebook design
- [The Holon Algebra: Operations Reference](/blog/primers/series-001-002-holon-ops/) — the full primitive set: bind, bundle, prototype, difference, accumulate, and more
- [Holon Memory: Subspaces and Engrams](/blog/primers/series-001-003-memory/) — incremental manifold learning, named memory snapshots, and the surprise fingerprint
- [Coordination-Free at Scale](/blog/primers/series-001-004-coordination-free/) — how eliminating the codebook eliminates an entire class of distributed systems problems

**Story** — the development arc, in order:

- [Prologue: The Idea That Wouldn't Leave](/blog/story/prologue/) — years of conversations and a Clojure conference video, before there was any code
- [Python: The First Two Weeks](/blog/story/series-002-001-first-experiments/) — twelve challenge batches, F1=1.000, and a performance ceiling that forced the Rust port

## The work

[**holon**](https://github.com/watmin/holon) *(Python)* — Where it started. The experimental lab that produced 17 challenge batches, F1=1.0 anomaly detection, and the encoding insight that made everything else possible.

[**holon-rs**](https://github.com/watmin/holon-rs) — The Rust implementation. Encoder, accumulator, subspace, engram memory. 12x faster than Python. SIMD-accelerated.

[**holon-lab-ddos**](https://github.com/watmin/holon-lab-ddos) — XDP + eBPF packet scrubber using Holon's engram memory for real-time DDoS mitigation. Zero-I/O functional pipeline. Rete-inspired compiled decision tree via sidecar.

[**holon-lab-baseline**](https://github.com/watmin/holon-lab-baseline) — LLM-driven traffic generation. Playwright + Ollama agents producing organic WordPress HTTP traffic for realistic experiment baselines.

[**algebraic-intelligence.dev**](https://github.com/watmin/algebraic-intelligence.dev) — This site.

Every file across all repositories is 100% LLM-generated. Zero hand-written code. Domain knowledge and architecture from the author; code from Grok, Sonnet, and Opus via prompting.
