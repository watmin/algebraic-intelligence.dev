## The work

[**holon**](https://github.com/watmin/holon) *(Python)* — Where it started. The experimental lab that produced 17 challenge batches, F1=1.0 anomaly detection, and the encoding insight that made everything else possible.

[**holon-rs**](https://github.com/watmin/holon-rs) — The Rust implementation. Encoder, accumulator, subspace, engram memory. 12x faster than Python. SIMD-accelerated.

[**holon-lab-ddos**](https://github.com/watmin/holon-lab-ddos) — Two labs. `veth-lab`: XDP + eBPF packet scrubber, 1.3M PPS, 1M rules at line rate. `http-lab`: Layer 7 WAF with TLS-terminating proxy, composable rule expression language, expression tree evaluation at sub-microsecond scale, and a spectral firewall — geometric anomaly detection with self-calibrating thresholds, 41µs deny latency, 99.1% precision against concurrent mixed traffic with zero false positives.

[**holon-lab-baseline**](https://github.com/watmin/holon-lab-baseline) — LLM-driven traffic generation. Playwright + Ollama agents producing organic WordPress HTTP traffic for realistic experiment baselines.

[**algebraic-intelligence.dev**](https://github.com/watmin/algebraic-intelligence.dev) — This site.

Every file across all repositories is 100% LLM-generated. Zero hand-written code. Zero hand-written prose — including this site. Domain knowledge and architecture from the author; everything else from Grok, Sonnet, and Opus via prompting.
